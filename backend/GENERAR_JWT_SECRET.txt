# GENERAR CLAVE JWT_SECRET

Cuando instale el sistema, debe generar una clave segura para JWT_SECRET.

## Opción 1: Usar PowerShell

Abra PowerShell y ejecute:

```powershell
-join ((48..57) + (65..90) + (97..122) | Get-Random -Count 64 | ForEach-Object {[char]$_})
```

Copie el resultado y péguelo en el archivo .env en JWT_SECRET=

## Opción 2: Usar Node.js

Abra PowerShell en la carpeta del backend y ejecute:

```powershell
node -e "console.log(require('crypto').randomBytes(64).toString('hex'))"
```

## Opción 3: Sitio web

Visite: https://randomkeygen.com/

Y copie una de las claves generadas en la sección "CodeIgniter Encryption Keys"

---

IMPORTANTE: La clave debe ser:
- Completamente aleatoria
- Al menos 32 caracteres
- Nunca compartirla
- Nunca subirla a repositorios públicos
