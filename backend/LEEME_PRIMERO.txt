                      [INICIO] INSTALACIÓN EN 5 PASOS
═══════════════════════════════════════════════════════════════════════

  1. Instalar Node.js v18+
  2. Descomprimir ZIP en C:\INEGI\Sistema\
  3. Renombrar .env.production → .env y configurar
  4. Ejecutar: npm install --production
  5. Ejecutar: .\start.bat

[IMPORTANTE] MARCO DEBE CAMBIAR EN EL ARCHIVO .env:

  [1] MONGO_URIRI
     Conexión a su MongoDB (local o Atlas)
     Ejemplo: mongodb://localhost:27017/INEGI_DB

  [2] JWT_SECRET
     Clave de seguridad (generar aleatoria de 64 caracteres)
     Ver: GENERAR_JWT_SECRET.txt

  [3] PUBLIC_BASE_URL
     URL de acceso al sistema
     Ejemplo: http://192.168.1.100:4000

PROBLEMAS COMUNES:

[X] MongoDB no conecta
   → Ver logs: pm2 logs inegi-sistema
   → Verificar MONGO_URI en .env
   → Probar con MongoDB Compass

[X] Puerto ocupado
   → Ver qué usa el puerto: netstat -ano | findstr :4000
   → Cambiar PORT en .env

[X] No accede desde otras PCs
   → Configurar firewall (ver README_INSTALACION.md)
   → Verificar PUBLIC_BASE_URL en .env

[X] Error al iniciar
   → Ver logs: pm2 logs inegi-sistema
   → Verificar .env configurado correctamente
   → Confirmar dependencias instaladas
